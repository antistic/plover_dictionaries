## Spelling Changes

### ^or -> ^our

```yaml
THOR: authorise
EPB/TKEFR: endeavour
EPB/TKEFRS: endeavours
TPAEUFR: favour
TPAUFR: favour
TPAEUFRT: favourite
HO*RPB: honour
HRAEUB: labour
HRAEURB: labour
```

### ^er -> ^re

```yaml
KAL/PW*ER: calibre
KAL/PW-R: calibre
STR*: centre # keeping STR: center for programming
```

### ^ize -> ^ise, ^ization -> ^isation

```yaml
AOEUZ: {^ise}
THORZ: authorise
KREUT/SAOEUZ: criticise
EFPL/SAOEUFD: emphasised
ORG: organise
RAELGS: realisation
RAELZ: realise
RAEFLD: realised
THRAELZ: realise that
R*EZ: recognise
REZ: recognise
```
```yaml
SA*EUGS: {^isation}
ORGS: organisation
```
